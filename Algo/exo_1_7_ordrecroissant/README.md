# Ordre croissant (2 nombres)

```

VARIABLES 

    a est un flottant
    b est un flottant

DEBUT PROGRAMME

    Lire a
    Lire b
    Si a > b 
    Alors
        Afficher b, a 
    Sinon Si a < b
    Alors
        Afficher a, b
    Sinon
       Afficher "Les 2 valeurs sont égales" 
    Fin Si


FIN PROGRAMME

```

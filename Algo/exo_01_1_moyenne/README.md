# Algorithme pour calculer la moyenne de 2 nombres

```

VARIABLES

    a est un entier
    b est un entier
    resultat est un flottant

DEBUT PROGRAMME 

    Afficher "Veuillez saisir 2 nombres séparés par un espace"
    Lire a
    Lire b 
    resultat <-- (a + b) / 2
    Afficher "Le résultat du calcul est : " , resultat

FIN PROGRAMME

```
